⏳ Verificando origen...
☁️ ¿Cómo habilitar config compartida entre PCs? (solo 1 vez)

Ejecutá este SQL en Supabase → SQL Editor:

create table if not exists gomatex_config (
  key text primary key,
  value text,
  updated_at timestamptz default now()
);
alter table gomatex_config enable row level security;
create policy "auth_read"  on gomatex_config for select using (auth.role() = 'authenticated');
create policy "auth_write" on gomatex_config for all    using (auth.role() = 'authenticated');

Después guardá los GIDs desde cualquier PC y todas las demás los verán automáticamente al ingresar.

Para obtener el GID de una pestaña: abrí la hoja → hacé clic en la pestaña → mirá la URL, el número después de gid= es el GID.
